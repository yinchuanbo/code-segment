在 TS 语法中使用映射类型，可以将原类型分别转换为`只读类型`和`可选类型`。

### 第一步：定义一个描述人员基本信息的简单接口

```ts
interface IPerson {
  name: string;
  age: number;
}
```

### 第二步：通过映射类型方法将原接口 IPerson 的属性转换为只读属性

```ts
type TReadonly<T> = {
  readonly [P in keyof T]: T[P];
};

type ReadonlyPerson = TReadonly<IPerson>;
```
